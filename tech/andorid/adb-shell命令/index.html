Andorid - Adb Shell命令

发表于： 2023-08-14 更新于： 2023-08-15 分类于： Andorid
字数：176 阅读：≈ 1分钟 浏览：

adb

# 获取 app 入口
$ adb logcat | grep -i displayed

shell

# 查看当前 activity
$ adb shell | grep -i display

# 获取当前页面的元素
$ adb shell dumpsys activity top
# 获取任务列表
$ adb shell dumpsys activity activities


$ aapt dump badging *.apk | grep launchable-activity
$ apkanalyzer 最新版本的 sdk 中才有

shell am（Activity Manager - 活动管理）

通过它可以启动App、启动Activity以及启动广播和服务等等。

$ adb shell am start -W com.esbook.reader/com.esbook.reader.activity.ActLoading
Starting: Intent { act=android.intent.action.MAIN cat=[android.intent.category.LAUNCHER] cmp=com.esbook.reader/.activity.ActLoading }
Status: ok
LaunchState: WARM
Activity: com.esbook.reader/.activity.ActLoading
TotalTime: 163
WaitTime: 164
Complete

参数介绍：

  • -S: 杀死应用后重新启动，相当于冷启动
