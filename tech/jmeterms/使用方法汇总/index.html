Jmeter&Ms - 使用方法汇总

发表于： 2023-07-11 更新于： 2023-08-13 分类于： Jmeter&Ms
字数：579 阅读：≈ 2分钟 浏览：

内置函数使用

参考地址：https://zhuanlan.zhihu.com/p/559416002

整型加法：${__intSum(${user_total},1)}

时间戳(13位)：${__time}

当前时间 ${__timeShift(,,,,)}

增加一天 ${__timeShift(,,P1d,,)}

增加一H ${__timeShift(,,PT1H,,)}

当前时间格式化：${__timeShift(yyyy-MM-dd HH:mm)} ==> 2023-07-26 18:53

当前时间格式化加一分钟：${__timeShift(yyyy-MM-dd HH:mm,,PT1M,,)} ==> 2023-07-26 18:54

相加函数（intSum）

使用场景：断言、参数计算

使用方法：${__intSum(${id2},1)}，其中 ${id2} 是从接口提取出来的值。代表计算 id2 增加 1 后的结果。

参数编写栏 有个编辑按钮，点击后可以在函数页面进行使用，支持预览结果以及函数配合使用。

注意

当 json 带转义转为字符串后，通过 ms 请求，\\\n 会被转义为 \n，注意检查下请求内容中的格式，否则接口数据提交时后端再次 json parse 时可能出错。如上，ms 中写请求体的时候，json 字符串中应修改换行为 \\\\\n
