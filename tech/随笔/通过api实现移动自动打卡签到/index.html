随笔 - 通过Api实现移动自动打卡签到

发表于： 2023-08-14 更新于： 2023-08-15 分类于： 随笔
字数：504 阅读：≈ 2分钟 浏览：

移动自动化签到；

签到送流量，连续签到赢 6 GB。

移动经常会送一些流量卡券到个人账户上，可以登录 APP 进入卡券页面使用。

最近推出连续签到赠送流量活动，最高 6GB，这也算是大额券了，哈哈，毕竟平时每月才固定160M，券到手后留着备用它不香吗？

参与方法

每天登录 APP，点击我的 -> 签到，完成签到即可。没啥难度，坚持就行。作为互联网小工，重复性强的工作必然要总结规律，自动化来实现。

本文通过接口实现每日自动签到，文中附源码地址，仅供学习参考，不得用于其他用途，如侵联删。

使用脚本

下载源码

github.com/abeelan/ha-ha

抓包

1、手机连接代理工具，使用账号密码登录，抓包请求websitepwdLogin参数列表：

  • ef：用户登录信息的加密字符串，粘贴到脚本的全局变量上；
  • ver：应用版本信息，这个可以忽略。

2、进入签到页面，抓包SignInH5请求：

  • 复制cookies信息填写到脚本内（不填也可以正常运行，最好保持与正常发送请求一致）；
  • 复制参数列表内的 token 值，填写到脚本的全局变量上。

运行

# 环境准备
$ pip install requests
$ pip install loguru

# 运行脚本
$ python mobile_sign.py

定时任务 (mac)

$ crontab -e
10 15 * * * python mobile_sign.py

查看每日执行结果有两种方式：

  • 通过 mail 命令，查看每日新邮件，包含详细日志；
  • 在脚本内指定 log 的输出路径。

完。
