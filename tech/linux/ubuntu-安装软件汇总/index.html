Linux - Ubuntu安装软件汇总

发表于： 2023-08-14 更新于： 2023-08-17 分类于： Linux
字数：791 阅读：≈ 2分钟 浏览：

基础命令

重启

$ reboot

查看 ip 地址

$ ip addr show

安装包

# 安装 ssh，已安装则忽略
$ sudo apt-get install ssh
# 输入密码

$ apt install net-tools  # ifconfig 命令

显示磁盘大小

# 查看硬盘剩余空间 -H
$ df -hl

# 查看硬盘的分区
$ sudo fdisk -l  

# 查看 IDE硬盘信息
$ sudo hdparm -i /dev/hda  

# 查看STAT硬盘信息
$ sudo hdparm -I /dev/sda  # 或 
$ sudo apt-get install blktool
$ sudo blktool /dev/sda id

# 查看目录占用空间
$ du -hs 目录名

# 优盘没法卸载
$ sync fuser -km /media/usbdisk

安装 adb

# 通过apt-get安装adb
$ sudo add-apt-repository ppa:nilarimogard/webupd8
$ sudo apt-get update
$ sudo apt-get install android-tools-adb

$ adb devices
* daemon not running; starting now at tcp:5037
* daemon started successfully
List of devices attached
1a34c478  unauthorized

# 未许可设备连接
# 重新开启「开发者选项 - USB 调试」，重启 adb server
# 在手机弹框中选择允许，即可成功连接

"注意：如果是未找到设备，那么继续向下看，以下内容未经实践，来源网络"

# 将 Android 设备连接至电脑，查看设备连接详情
$ lsusb
Bus 002 Device 002: ID 8087:8000 Intel Corp. Integrated Rate Matching Hub
Bus 002 Device 001: ID 1d6b:0002 Linux Foundation 2.0 root hub
Bus 001 Device 002: ID 8087:8008 Intel Corp. Integrated Rate Matching Hub
Bus 001 Device 001: ID 1d6b:0002 Linux Foundation 2.0 root hub
Bus 004 Device 001: ID 1d6b:0003 Linux Foundation 3.0 root hub
Bus 003 Device 004: ID 1a2c:2c27 China Resource Semico Co., Ltd USB Keyboard
Bus 003 Device 003: ID 046d:c077 Logitech, Inc. M105 Optical Mouse
"Bus 003 Device 002: ID 18d1:4ee9 Google Inc. Nexus/Pixel Device (MIDI + debug)"
Bus 003 Device 001: ID 1d6b:0002 Linux Foundation 2.0 root hub


# 创建 adb_usb.ini 文件并写入 id
$ echo 0x18d1 > ~/.android/adb_usb.ini

# 添加权限
$ sudo vim /etc/udev/rules.d/70-android.rules
# 加入以下内容，注意红色部分对应的就是lsusb得到的id
SUBSYSTEM=="usb", ATTRS{idVendor}=="18d1", ATTRS{idProduct}=="1002", MODE="0666"

# 重启 USB 服务
$ sudo chmod a+rx /etc/udev/rules.d/70-android.rules
$ sudo service udev restart

# 重启adb服务
$ adb kill-server
$ sudo adbstart-server
$ adb devices
Listof devices attached
1a34c478    device

安装 SVN

要在 Ubuntu 上安装 SVN，请按照以下步骤进行操作：

$ sudo apt-get install subversion
$ svn --version
$ svn co --username your_username --password your_password https://svn.example.com/svn/myproject

设置静态 IP 地址

# 修改静态 IP 地址
$ vim /etc/netplan/00-installer-config.yaml
# This is the network config written by 'subiquity'
network:
  ethernets:
    enp2s0:
      dhcp4: no
      optional: true
      addresses: [192.168.1.222/24] # 配置的静态ip地址和掩码
      routes:
        - to: default
          via: 192.168.1.1  # 网关地址
      nameservers:
        addresses: [114.114.114.114,8.8.8.8]
  version: 2
  
# 查看网络服务状态
$ sudo networkctl status

# 重启网卡，加载配置
$ sudo netplan apply
