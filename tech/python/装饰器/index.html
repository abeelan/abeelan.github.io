内置装饰器

@classmethod 类方法

# 有如下格式化日期的类 接收 年月日 三个参数
class DateFmt:
    def __init__(self, year, month, day):
        self.year = year
        self.month = month
        self.day = day

    def output(self):
        print(f"{self.year} 年 {self.month} 月 {self.day} 日")

d = DateFmt(2022, 10, 28)
d.output()

当接收日期的参数变更为字典时，以上类就不支持了。但是该类已经在使用中，为不影响其他调用该类的代码，需要设置一个类方法，来实现接收字典类型的参数，完成输出日期的功能，实现如下。

class DateFmt:
    def __init__(self, year, month, day):
        self.year = year
        self.month = month
        self.day = day

    def output(self):
        print(f"{self.year} 年 {self.month} 月 {self.day} 日")

    @classmethod
    def json_fmt(cls, d: dict):
        year, month, day = d.values()
        return cls(year, month, day)  # 返回该类的普通调用实例
        

json_date = {"year": 2022, "month": 10, "day": 28}
DateFmt.json_fmt(json_date).output()

类不需要初始化就可以调用类方法。需要注意的是，类方法内只能调用类变量和类方法，不能调用实例变量和实例方法。

只有在加载类时，类方法才会被加载到内存中。

@staticmethod 静态方法

此方法没有用到任何实例方法或变量、类方法或变量，可以作为一个单独的函数来使用。

这种情况下，编译器会出现波浪线提示，则可以使用该装饰器取消提示。

class Demo:
     
    @staticmethod
    def start():  # 这里的 self 就可以不写了
        print("game start")

程序运行开始，就会把静态方法加载到内存中。

闭包函数

函数引用

def demo():
  print("haha")

# 函数可以被赋值给一个变量
haha = demo
haha()  # haha

闭包函数

  • 闭包的内部函数中，可以对外部作用域的变量进行引用
  • 闭包无法修改外部函数的变量值
  • 闭包可以保存当前的运行环境
# 实现
def stu_grade(grade):
    def output(name, gender):
        # grade 读取的是上层函数的形参
        print(f"name={name}, gender={gender}, grade={grade}")
    return output

grade1 = stu_grade(1)  # 保存 1 年级，grade 无需重复声明
# print(grade1)  # <function stu_grade.<locals>.output at 0x10b9b1ea0>
grade1("tom", "M")  # name=tom, gender=M, grade=1

可以看到，stu_grade 传入为 1 年级后，grade1 其实就是 output 函数的引用，可以直接添加括号，传入 name 和 gender 打印出信息。

装饰器

增强代码可读性，可维护性

def timer(func):
    # 装饰器会有一个默认参数，参数就是被装饰函数的引用
    def inner():
        print("setup")
        func() 
        print("teardown")
    return inner


@timer
def output():
    print("hello world!")

output()

实现一个装饰器，计算函数的执行时间。

from datetime import datetime

def timer(func):
    def inner():
        start = datetime.now()
        func()
        end = datetime.now()
        print(f"函数执行时间为：{end-start}")
    return inner


@timer
def output():
    print("hello world!")

带参数装饰器

from datetime import datetime

def timer(func):
    # func 的形参在 inner 方法接收 因为不确定传参个数及类型，所以使用不定长形参
    def inner(*args, **kwargs):
        start = datetime.now()
        func(*args, **kwargs)
        end = datetime.now()
        print(f"函数执行时间为：{end - start}")
    return inner

@timer
def hello(name):
    print(f"hello {name}!")
