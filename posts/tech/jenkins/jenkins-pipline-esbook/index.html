2、安装 allure#

安装 node#

先安装 node，未安装 node 直接启动 allure 会报错：/usr/bin/env: node: No such file or directory

# 创建安装目录
$ mkdir node && cd node
# 下载安装包
$ wget https://npm.taobao.org/mirrors/node/v15.8.0/node-v15.8.0-linux-x64.tar.gz
# 解压缩安装包
$ tar -zxvf node-v15.8.0-linux-x64.tar.gz
# 创建软链
$ ln -s /data/node/node-v15.8.0-linux-x64/bin/npm /usr/bin/npm
$ ln -s /data/node/node-v15.8.0-linux-x64/bin/node /usr/bin/node
安装 allure#

方法一

$ npm install -g allure-commandline --save-dev

方法二

下载包推送到服务器上解压

下载地址：https://registry.npmjs.org/allure-commandline/-/allure-commandline-2.13.0.tgz

# 上传到服务器上，-be 是防止乱码卡死
$ rz -be 
$ tar -zxvf allure-commandline-2.13.0.tgz
$ ln -s /data/allure/package/bin/allure /usr/bin/allure

3.1、 创建项目

3.2、 配置项目

  • 参数化构建过程：用于选择脚本执行环境

  • 添加流水线脚本

pipeline {

    agent any

    stages {
        stage('拉取代码'){
            steps {
                // 拉取代码，可选 svn、git 等，通过 pipline 语法生成
                // 只需要添加仓库 URL 和 证书 即可，其他默认就行
            }
        }
        stage('验证环境'){
            steps {
                // 安装运行环境
                sh "pip3 install -r requirements.txt"
                // 验证测试环境
                sh "chmod 777 ./ci/env.sh"
                sh "sh ci/env.sh"
            }
        }
        stage('运行测试'){
            steps{
                sh "python3 run.py ${env.choice}"
            }
        }
        stage('生成报告'){
            steps{
                script{  
                // 集成allure
                allure includeProperties: false, jdk: '', report: 'reports/allure-report', results: [[path: 'reports/allure-result']]
            }
            }
        }
   }
}
