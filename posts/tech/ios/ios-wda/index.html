WebDriverAgent#

用于远程控制 iOS 设备，是对 APP 进行自动化测试的完美工具。它通过链接XCTest.framework和调用 Apple 的 API 来直接在设备上执行命令。

1⃣️ 下载项目

# https://github.com/appium/WebDriverAgent
$ git clone https://github.com/appium/WebDriverAgent

2⃣️ 构建到设备

打开 WebDriverAgent.xcodeproj并开始WebDriverAgentRunner测试

报错解决#
